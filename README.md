# Save on Time

Save on Time est un utilitaire Windows permettant de planifier des copies automatiques de fichiers via le Planificateur de tâches Windows.

## Fonctionnalités

- **Copie directe** de dossiers avec barre de progression
- **Tâches planifiées** : choisissez les jours et l'heure d'exécution
- **Gestion des tâches** : visualisez et supprimez les tâches planifiées existantes

## Installation

Téléchargez l'installeur depuis la [page du projet](https://radiant-maxine.github.io/Save_On_Time/).

## Utilisation

L'application nécessite des **droits administrateur** pour créer et gérer les tâches planifiées Windows.

1. Sélectionnez un répertoire source et un répertoire de destination
2. Effectuez une copie directe ou planifiez une tâche automatique
3. Gérez vos tâches planifiées depuis la section dédiée
